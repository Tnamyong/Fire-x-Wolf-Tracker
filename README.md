The two Data Pack zip files (Data pack 1, Data pack 2) contain the data I used for this project, sourced from various government
and environmental online sources like the EPA, Census.gov, and Washington Deparment of Natural Resources, The helper zip file (helpers)
contains different helper functions that are used to make my visualization (Fire x Wolf map) in QGIS, like my borders and buffer zones. 
The qgz file "fxw.qgz" will allow you to load my project into QGIS. And my finished visualization is "Fire x Wolf map" which has the various
forest fires and wolf populations mapped out. 

I chose to use QGIS for this project as ArcGIS is blocked behind a paywall of a price too much for me to pay, and QGIS is a great
alternative for those wanting to learn Geospatial Information Systems.
